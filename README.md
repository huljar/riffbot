# Installation

```bash
# make a virtualenv somewhere
python3 -m venv /path/to/venv/riffbot
source /path/to/venv/riffbot/bin/activate
# install wheel, then requirements
pip3 install -U wheel
pip3 install -U -r requirements.txt
```
